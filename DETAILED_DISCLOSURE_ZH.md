# Inside The System - 简体中文汉化补丁

## 项目说明
这是 Minecraft 模组 **Inside The System** 的官方简体中文本地化版本，由原作者 **Dreg Iron** 授权进行非商业性汉化。

## 功能特性

### 主要功能
- 完整的简体中文本地化（包括物品、方块、实体、GUI、对话文本）
- 修正了原版部分 `.class` 文件以正确显示中文
- 保留原版所有游戏机制和剧情内容

### 游戏内容
这是一个以剧情为导向的冒险模组，包含：
- 多个自定义维度
- 新的方块、物品和实体
- 完整的任务系统和对话系统
- 多个结局分支
- 恐怖/悬疑主题的游戏体验

## ⚠️ 技术说明（重要）

### 服务器命令执行
本模组在特定条件下会通过 Minecraft 服务器命令接口执行游戏内操作，用于实现剧情演出效果。这些操作**仅在模组剧情触发时自动执行**，不会在玩家正常游戏时随意运行。

#### 执行的命令类型：

**1. 显示标题/字幕**
```
/title @a times <delay> <display> <exit>
/title @a title <text>
/title @a subtitle <text>
/title @a actionbar <text>
```
- **用途：** 剧情对话和场景提示
- **影响范围：** 所有在线玩家
- **触发时机：** 特定剧情节点、任务完成时

**2. 传送玩家**
```
tp @a <x> <y> <z>
```
- **用途：** 剧情强制传送
- **影响范围：** 所有玩家
- **触发时机：** 进入特定维度或完成关键剧情

**3. 设置重生点**
```
spawnpoint @a <x> <y> <z>
```
- **用途：** 设置剧情检查点
- **影响范围：** 所有玩家
- **触发时机：** 进入安全区域（如"家"维度）

**4. 修改时间**
```
time set night
time set day
```
- **用途：** 营造剧情氛围
- **影响范围：** 整个服务器世界
- **触发时机：** 特定任务阶段

**5. 放置方块**
```
/setblock <x> <y> <z> <block>
```
- **用途：** 自动建造剧情场景
- **影响范围：** 指定坐标
- **触发时机：** 任务进行中或场景初始化

**6. 停止音效**
```
/stopsound
```
- **用途：** 控制背景音乐和音效
- **影响范围：** 所有玩家
- **触发时机：** 剧情转折点

**7. 踢出玩家**
```
kick @a <reason>
```
- **用途：** 触发特殊结局时踢出玩家
- **影响范围：** 所有玩家
- **触发时机：** 特定坏结局（如"结局F"）

---

### 安全说明

1. **仅限剧情触发：** 所有命令执行都绑定在特定的剧情事件上，玩家正常游戏时不会自动运行
2. **服务器权限：** 需要服务器开启命令方块或给予模组执行命令的权限（通常为 OP 等级 4）
3. **可预测性：** 所有命令执行都在剧情流程中明确标注，玩家可以提前知晓
4. **无数据修改：** 不修改玩家存档文件，不访问外部网络
5. **无恶意行为：** 所有命令仅用于模组剧情演出，不会删除建筑、物品或进行其他破坏性操作

## 安装说明

1. 确保已安装 **Minecraft** (推荐 1.20.1 版本)
2. 安装 **NeoForge** (对应 Minecraft 版本)
3. 将本模组放入 `mods` 文件夹
4. 原版 Inside The System 模组**不需要**同时安装（本版本已包含全部内容）
5. 启动游戏即可体验完整中文剧情

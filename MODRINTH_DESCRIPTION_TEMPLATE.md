## ⚠️ 技术功能披露（Modrinth 审核要求）

**本模组包含以下系统级交互功能，作为剧情体验的一部分：**

### 1. IP 地址记录
- **功能：** 在特定剧情阶段获取您的 IP 地址和地理位置
- **数据发送：** 发送到第三方 API（ipify.org 和 ip-api.com）
- **控制方法：** 使用命令 `/safeip` 可禁用此功能（改用随机位置数据）
- **用途：** 在游戏内告示牌上显示位置相关信息

### 2. 桌面文件创建（仅 Windows）
- **行为：** 在特定结局（特别是"结局F"）期间在桌面创建文件
- **文件：** 最多 100 个文本文件，格式为 "I am coming for you_000.txt"
- **附加：** 可能创建/更新 "code.txt" 包含特定代码

### 3. 更改桌面壁纸（仅 Windows）
- **行为：** 在特定剧情事件中自动更改 Windows 桌面壁纸
- **技术：** 使用 PowerShell 和 Windows API（SystemParametersInfo）
- **图片：** 使用模组内置的 "story.png"

### 4. 执行外部程序（仅 Windows）
- **行为：** 启动外部程序（PowerShell 脚本）以增强沉浸感
- **内容：** 播放视频文件（Knock.mp4）并模拟键盘输入
- **权限：** 需要 PowerShell 执行权限

### 5. 服务器命令执行
- **行为：** 在剧情推进时自动执行 Minecraft 服务器命令
- **命令类型：** /title, /tp, /setblock, /time set, /kick, /stopsound 等
- **影响：** 影响服务器所有玩家，用于 cinematic 效果

**重要提示：**
- 这些功能是原模组**故意设计**的"打破第四面墙"体验
- 仅由特定游戏内剧情事件触发
- 除 IP 记录外，**无法禁用**其他功能
- 这是原模组的核心设计，汉化版本保留其全部内容

## 🛠️ 构建说明

这是反编译的 Minecraft 模组。如需重新编译：

1. 设置 NeoForge 1.20.1 开发环境
2. 将 `extracted/` 内容复制到项目的 `src/main/resources/`
3. 使用 Maven 或 Gradle 编译

**注意：** 源码使用 CFR 0.152 反编译。变量名和部分结构可能与原始源码不同，但逻辑功能完全等价。详细说明见 `SOURCE_CODE_INFO.md`。

## ⚠️ Technical Features Disclosure

**This mod includes system-level interactions as part of the narrative:**

1. **IP Address Logging** - Can be disabled with `/safeip` command
2. **Desktop File Creation** (Windows only) - Creates files on user's desktop
3. **Wallpaper Change** (Windows only) - Changes desktop background
4. **External Program Execution** (Windows only) - Launches PowerShell scripts
5. **Server Commands** - Executes Minecraft commands for cinematic effects

**See [FULL_TECHNICAL_DISCLOSURE_ZH.md](FULL_TECHNICAL_DISCLOSURE_ZH.md) for complete details.**

## 🛠️ Building

This is a decompiled Minecraft mod. To rebuild:

1. Set up a NeoForge 1.20.1 development environment
2. Copy the `extracted/` contents into your project's `src/main/resources/`
3. Compile with Maven/Gradle

**Note:** The source code was decompiled using CFR 0.152. Variable names and some structure may differ from original source, but logic is functionally equivalent.

## 📝 About This Localization

This is an **unofficial Chinese language pack** for Inside The System. It provides:

- Complete Simplified Chinese translation for all in-game text
- Fixed character display issues in original .class files
- **Enhanced texture:** `aiko.png` (entity texture) - visually improved for better aesthetics
- All original game mechanics and story content preserved
- No gameplay modifications beyond language translation and texture enhancement

**This is not the original mod.** You still need to understand English to some extent for certain UI elements that use system fonts.
